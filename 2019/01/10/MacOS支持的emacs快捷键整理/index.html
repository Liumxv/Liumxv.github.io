MacOS支持的emacs快捷键整理

2019.01.10 MacOS

光标移动

  • C + f : 光标右移一位
  • C + b : 光标左移一位
  • C + p : 光标上移一位
  • C + n : 光标下移一位

文本选择

  • C + shift + f : 选中右一文本 == shift + →
  • C + shift + b : 选中左一文本 == shift + ←
  • C + shift + p : 选中这行到上一行这个位置的文本 == shift + ↑
  • C + shift + n : 选中这行到下一行这个位置的文本 == shift + ↓
  • C + shift + a : 选中当前到行首的文本 == shift + cmd + ←
  • C + shift + e : 选中当前到行尾的文本 == shift + cmd + →

文本编辑

  • C + a : 光标移动到行首
  • C + e : 光标移动到行尾
  • C + h : 删除前一文本 == delete
  • C + o : 回车键 == return
  • C + t : 交换光标两侧的文本
  • C + L : 跳转到光标所在行
